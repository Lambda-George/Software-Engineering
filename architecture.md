Web Application
  The web application will provide an interface to the end user, either student or teacher.
  Allow for the user to view both the full timetable and upcoming events.
  
Backend
  The backend will control what will be displayed on the final system.
  The backend will also be responsible for the notifications sent to students when a deadline is approaching. 

Mobile App
  The mobile app will display the web applications in a mobile friendly structure.
  The mobile app will be capable of sending deadline notifications via the mobile devices dropdown menu.
